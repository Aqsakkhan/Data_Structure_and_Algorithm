# 🧠 DSA Pattern-Wise Question List (All Core Topics)

This repository contains **Data Structures & Algorithms problems organized pattern-wise**.  
The goal is to master **problem-solving patterns** instead of memorizing solutions.

---

## 🔹 1. Sliding Window Pattern
**Used for:** Subarrays / substrings (contiguous)

### Questions:
- Maximum sum subarray of size K  
- First negative number in every window of size K  
- Count anagrams of a string  
- Longest substring without repeating characters  
- Longest substring with at most K distinct characters  
- Smallest subarray with sum ≥ K  
- Sliding window maximum  
- Longest repeating character replacement  
- Minimum window substring  

---

## 🔹 2. Prefix Sum + HashMap Pattern
**Used for:** Subarray sum problems

### Questions:
- Subarray sum equals K  
- Longest subarray with sum K  
- Count subarrays with sum 0  
- Subarray sum divisible by K  
- Longest subarray with equal 0s and 1s  
- Equilibrium index  
- Count subarrays with given sum  
- Continuous subarray sum

## 🔹 3. Two Pointers Pattern
**Used for:** Pairs, optimization, sorted data

### Questions:
- Reverse array / string  
- Check palindrome  
- Two sum (sorted array)  
- Remove duplicates from sorted array  
- Container with most water  
- Merge two sorted arrays  
- 3Sum  
- 4Sum

- Intersection of two arrays  
- Majority element
